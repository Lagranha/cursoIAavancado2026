Construtor de Prompts XML — via Anthropic API

Preencha os campos abaixo para construir um prompt estruturado em XML. O formulário montará as tags automaticamente e enviará à API do Claude. O <call_to_action> é o texto principal que será analisado pelo assistente.

<situação> Contexto do problema
Descreva o cenário, contexto ou problema que motiva essa tarefa.
<papel> Papel do assistente
Qual personagem, especialista ou função o assistente deve assumir?
<tarefa> Tarefa principal
O que exatamente o assistente deve fazer? Seja específico e direto.
<formato> Formato de saída
Como a resposta deve ser estruturada? (tabela markdown, JSON, lista, texto corrido…)
<restrições> Regras e limitações
O que o assistente NÃO deve fazer? Quais regras deve seguir?
<critérios> Critérios de qualidade
Como uma boa resposta se parece? Quais são os critérios de avaliação?
<exemplos> Exemplos de entrada/saída opcional
Forneça 1–3 exemplos de entrada e saída esperada para guiar o modelo.
<call_to_action> Texto a ser analisado
Cole aqui o conteúdo que o assistente deve processar — e-mail, mensagem, documento, etc.
